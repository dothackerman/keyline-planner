# 12. Glossary

| Term | Definition |
|------|-----------|
| **AOI** | Area of Interest — the geographic extent (polygon or bounding box) for which contours are generated |
| **COG** | Cloud Optimized GeoTIFF — a GeoTIFF with internal tiling and overviews for efficient partial access |
| **Contour line** | An isoline connecting points of equal elevation on a surface |
| **CRS** | Coordinate Reference System — defines how coordinates map to locations on Earth |
| **DEM** | Digital Elevation Model — a raster grid of ground surface elevations |
| **EPSG:2056** | The EPSG code for Swiss LV95 / CH1903+ projected coordinate system |
| **EPSG:4326** | The EPSG code for WGS 84 geographic coordinate system (lat/lon) |
| **GDAL** | Geospatial Data Abstraction Library — open-source raster/vector translator and processing engine |
| **GeoJSON** | An open standard format for encoding geographic data structures in JSON |
| **GeoPackage** | An OGC standard format based on SQLite for storing vector/raster data |
| **GSD** | Ground Sample Distance — the physical size of one pixel in the raster (e.g., 0.5m or 2.0m) |
| **Keyline design** | A landscape design technique using contour patterns to manage water flow and soil conservation |
| **LV95** | Swiss national coordinate system (CH1903+ / Landesvermessung 1995) |
| **Mosaic** | Combining multiple raster tiles into a single seamless raster |
| **STAC** | SpatioTemporal Asset Catalog — a specification for cataloguing geospatial data assets |
| **swissALTI3D** | High-precision digital elevation model of Switzerland published by swisstopo |
| **swisstopo** | Federal Office of Topography — Swiss national mapping agency |
| **VRT** | GDAL Virtual Format — a lightweight mosaic definition that references source rasters without copying data |
